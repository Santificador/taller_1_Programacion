## 🚀 Inicio en 3 Pasos

### Opción A: QUARKUS (Inmediato) ⭐

```bash
# Paso 1
cd venta-cafe-api

# Paso 2
docker-compose up --build

# Paso 3
Abrir navegador: http://localhost:8080/index.html
```

**¡Listo en 5 minutos!** ✅

### Opción B: JHIPSTER (Profesional) ⭐

```bash
# Paso 1
cd venta-cafe-jhipster
./generate.bat  # (Windows) o ./generate.sh (Linux)

# Paso 2: Después de la generación
cd venta-cafe
mvn spring-boot:run  # Terminal 1
npm start             # Terminal 2

# Paso 3
Abrir navegador: http://localhost:3000
Login: admin / admin
```

**¡Listo en 20 minutos!** ✅

## ❓ Preguntas Frecuentes

**¿Qué versión debo usar?**
→ Quarkus si necesitas algo YA. JHipster si quieres algo PROFESIONAL.

**¿Puedo usar ambas?**
→ Sí, son independientes. Ambas usan PostgreSQL pero con DBs diferentes.

**¿Hay que pagar?**
→ No, ambas son 100% open source y gratuitas.

**¿Cuál es más fácil?**
→ Quarkus es más simple. JHipster es más robusto pero con más curva.

**¿Puedo migrar Quarkus → JHipster?**
→ Sí, exporta datos de DB e importa en la nueva versión.

**¿Necesito aprender mucho?**
→ No, ambos están muy documentados con ejemplos completos.
